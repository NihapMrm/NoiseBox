How sounds work

Noisebox doesn't ship a bloated audio library. Sounds resolve in three tiers.

  1. Bundled. A small core set ships with the app and works instantly, offline.
  2. Cached. Anything you've used before is stored locally and loads instantly next time.
  3. Downloaded on demand. New sounds are fetched from Freesound or Pixabay using your own free API key, then cached for offline use.

Add a free Freesound or Pixabay API key in settings to unlock the full library.
